# Deployment Scripts

Automation scripts for setting up and deploying Kraken D0010.

## Scripts

- `setup_system_dependencies.sh` - System packages
- `setup_python_environment.sh` - Python virtual environment
- `setup_databases.sh` - Database configuration
- `validate_deployment.sh` - Deployment verification
- `quick_deploy.sh` - One-command deployment orchestrator

## Usage

Run `quick_deploy.sh` for automated setup, or run individual scripts in order for manual deployment.
